# 安装jdk

  1. 下载jdk8（别下错了，64位系统不要下载32位jdk）： 地址：https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html

  2. 创建jdk放置的目录 sudo mkdir /usr/lib/jvm

  3. 解压到该目录 sudo tar -zxvf jdk-8u291-linux-x64.tar.gz -C /usr/lib/jvm

  4. 修改环境变量 sudo vi ~/.bashrc

在文件最底下粘贴： #set oracle jdk environment export JAVA_HOME=/usr/lib/jvm/jdk1.8.0_291 export JRE_HOME=${JAVA_HOME}/jre export CLASSPATH=.😒{JAVA_HOME}/lib:${JRE_HOME}/lib export PATH=${JAVA_HOME}/bin:$PATH

  1. 使环境变量生效 source ~/.bashrc

  2. 系统注册此jdk sudo update-alternatives --install /usr/bin/java java /usr/lib/jvm/jdk1.8.0_291/bin/java 300

  3. 查看是否安装成功 java -version

  4. 如果安装多版本jdk，可以使用下面命令做切换 sudo update-alternatives --config java

# ubuntu上使用ssh

参考地址 (opens new window)

如果你只是想登陆别的机器的SSH只需要安装openssh-client（ubuntu有默认安装，如果没有则sudo apt-get install openssh-client），如果要使本机开放SSH服务就需要安装openssh-server。

  1. 查看当前的ubuntu是否安装了ssh-server服务。默认只安装ssh-client服务。 dpkg -l | grep ssh

  2. 安装ssh-server服务 sudo apt-get install openssh-server

  3. 查看ssh是否安装完成 dpkg -l | grep ssh

  4. 查看ssh-server是否启动 ps -e | grep ssh

# ubuntu中安装docker

\1. 检查linux版本内核和存储驱动> uname -a> ls -l /sys/class/misc/device-mapper \2. 通过官方shell脚本安装> sudo apt-get install curl安装curl时, 可能需要卸载其他软件, 使用apt-get --purge remove software_name curl -sSL https://get.docker.com/ (opens new window) | sudo sh \3. 添加docker用户组> sudo groupadd docker> sudo gpasswd -a ${USER} docker> sudo service docker restart

# 检查linux版本内核和存储驱动

> uname -a
> ls -l /sys/class/misc/device-mapper

# 通过官方shell脚本安装

下载官方安装脚本需要使用curl

> sudo apt-get install curl
# 安装curl时, 可能需要卸载其他软件, 使用apt-get --purge remove  software_name

下载官方安装脚本

> curl -sSL https://get.docker.com/ | sudo sh

# 安装后添加docker用户组

> sudo groupadd docker
> sudo gpasswd -a ${USER} docker
> sudo service docker restart

# 安装gcc和make

基础镜像什么都没有带，如果要编译c相关的工具，需要这两个工具，接下来是安装步骤：

  1. sudo apt install make
  2. sudo apt install make-guile
  3. sudo apt-get install gcc

也可以直接安装基础软件包： sudo apt install build-essential
